# 手机端网络连接配置说明

## 问题说明
手机端无法访问局域网IP `10.216.82.205:8000`，需要根据使用场景选择正确的配置。

## 配置方案

### 方案1：HBuilderX内置浏览器调试
**推荐使用**，最简单的方式：
1. 直接在HBuilderX中运行到内置浏览器
2. 配置使用 `localhost:8000`（已默认配置）

### 方案2：微信小程序调试
1. 打开微信开发者工具
2. 在详情 → 本地设置中勾选"不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书"
3. 配置使用 `localhost:8000`

### 方案3：真机调试（手机）
需要确保：
1. 手机和电脑连接**同一WiFi网络**
2. 关闭电脑防火墙，或允许8000端口
3. 查看电脑的局域网IP地址：
   - Windows：打开cmd，输入 `ipconfig`，找到IPv4地址（如 `192.168.1.100`）
4. 修改 `config/index.js` 中的配置：

```javascript
const isDev = true

const config = {
    dev: {
        baseUrl: 'http://192.168.1.100:8000'  // 改成你的电脑IP
    },
    // ...
}
```

## 检查后端服务
确保后端服务正在运行：
```bash
cd 后端
python main.py  # 或其他启动命令
```

访问 http://localhost:8000 确认服务正常。

## 常见问题

**Q：手机连接超时怎么办？**
A：检查以下几点：
1. 手机和电脑是否在同一WiFi
2. 电脑IP地址是否正确
3. 防火墙是否关闭
4. 后端服务是否正常运行

**Q：如何快速切换配置？**
A：修改 `config/index.js` 中的 `baseUrl` 即可，不需要重启项目。
