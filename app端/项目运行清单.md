# 项目运行清单

## 项目概述
这是一个基于 uni-app 框架开发的大学生职业规划智能体 App 端项目，支持小程序、H5、App 多端运行。

## 环境要求
- HBuilderX 3.6+ 或 VS Code + uni-app 插件
- Node.js 14+（如需使用 npm）
- 微信开发者工具（小程序开发）
- Android Studio / Xcode（App 原生开发）

## 必须修改的配置

### 1. 网络地址配置（重要！）
**文件位置：** `config/index.js`

```javascript
const isDev = true

const config = {
    dev: {
        baseUrl: 'http://localhost:8000'  // 开发环境地址
    },
    prod: {
        baseUrl: 'https://your-production-domain.com'  // 生产环境地址（需要修改）
    },
    lan: {
        baseUrl: 'http://10.216.80.152:8000'  // 局域网地址（需要修改为你的IP）
    }
}

export default {
    baseUrl: isDev ? config.lan.baseUrl : config.prod.baseUrl,  // 当前使用的环境
    
    setBaseUrl(url) {
        this.baseUrl = url
    }
}
```

**需要修改的内容：**
- `lan.baseUrl`：修改为你电脑的局域网 IP 地址（手机真机调试时需要）
- `prod.baseUrl`：修改为你的生产环境域名（正式上线时需要）
- `isDev`：根据需要切换开发/生产环境

**如何获取局域网 IP：**
- Windows：打开 cmd，输入 `ipconfig`，查看 IPv4 地址
- macOS/Linux：打开终端，输入 `ifconfig` 或 `ip addr`

### 2. 小程序 AppID 配置
**文件位置：** `manifest.json`

```json
"mp-weixin": {
    "appid": "",  // 微信小程序 AppID（需要修改）
    "setting": {
        "urlCheck": false,
        "es6": true,
        "postcss": true,
        "minified": true
    },
    ...
}
```

**需要修改的内容：**
- `appid`：填入你的微信小程序 AppID（如果要发布到微信小程序）

### 3. H5 代理配置（可选）
**文件位置：** `manifest.json`

```json
"h5": {
    "title": "职业规划系统",
    "router": {
        "mode": "hash",
        "base": "./"
    },
    "devServer": {
        "port": 8080,
        "disableHostCheck": true,
        "proxy": {
            "/api": {
                "target": "http://localhost:8000",  // 代理目标地址（需要修改）
                "changeOrigin": true,
                "secure": false,
                "pathRewrite": {
                    "^/api": ""
                }
            }
        }
    }
}
```

**需要修改的内容：**
- `proxy./api.target`：修改为你的后端地址（H5 开发时使用）

## 后端服务要求
App 端需要配合后端服务使用，请确保：
1. 后端服务已启动并运行在配置的地址上
2. 后端端口为 8000（或修改为你的端口）
3. 后端 CORS 配置允许跨域请求
4. 手机和电脑在同一局域网（真机调试时）

## 运行步骤

### 1. 微信小程序运行
1. 打开 HBuilderX
2. 导入项目：文件 → 导入 → 从本地目录导入
3. 运行：运行 → 运行到小程序模拟器 → 微信开发者工具
4. 在微信开发者工具中查看和调试

### 2. H5 运行
1. 打开 HBuilderX
2. 运行：运行 → 运行到浏览器 → Chrome
3. 在浏览器中查看和调试

### 3. App 真机运行
1. 打开 HBuilderX
2. 连接手机到电脑（开启 USB 调试）
3. 运行：运行 → 运行到手机或模拟器 → 选择你的设备
4. 在手机上查看和调试

## 常见问题

### Q: 手机端无法连接后端？
A: 请检查：
1. 手机和电脑是否在同一局域网
2. `config/index.js` 中的 `lan.baseUrl` 是否为电脑的正确 IP
3. 后端服务是否已启动
4. 电脑防火墙是否阻止了 8000 端口

### Q: 小程序提示 "request:fail"？
A: 请检查：
1. `manifest.json` 中 `mp-weixin.appid` 是否正确配置
2. 微信开发者工具中是否关闭了 "不校验合法域名"
3. 后端服务是否正常运行

### Q: H5 跨域问题？
A: 请检查：
1. `manifest.json` 中 H5 的 proxy 配置是否正确
2. 后端是否配置了 CORS
