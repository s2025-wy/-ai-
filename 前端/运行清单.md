# 前端运行清单

## 项目概述

本项目是基于AI的大学生职业规划智能体的前端应用，使用Vue 3 + TypeScript构建，提供用户界面、岗位探索、学生画像、职业规划、智能问答等功能。

## 技术栈

- Vue 3 + TypeScript
- Pinia (状态管理)
- Vue Router (路由)
- Element Plus (UI组件库)
- ECharts (数据可视化)
- Axios (HTTP请求)
- XLSX (Excel文件处理)
- HTML2PDF.js (PDF生成)

## 环境配置

### 1. 安装Node.js

确保安装Node.js 20.19.0或更高版本。

### 2. 安装依赖

```bash
# 进入前端目录
cd 前端/vue-A13

# 安装依赖
npm i
```

### 3. 配置环境变量

前端项目默认连接到 `http://localhost:8000` 的后端服务。如果后端服务运行在不同地址，需要修改 `src/api` 目录下的API请求配置。

## 运行步骤

### 开发环境

1. 进入前端目录：`cd 前端/vue-A13`
2. 启动开发服务器：
   ```bash
   npm run dev
   ```
3. 访问 `http://localhost:5173` 查看应用

### 生产环境

1. 进入前端目录：`cd 前端/vue-A13`
2. 构建项目：
   ```bash
   npm run build
   ```
3. 部署 `dist` 目录到服务器

## 核心功能

1. **用户认证**：登录、注册功能
2. **岗位探索**：岗位搜索、筛选、申请、收藏
3. **学生画像**：个人信息管理、简历上传、能力分析
4. **职业规划**：职业路径规划、行动计划制定
5. **智能问答**：基于大模型的职业咨询
6. **知识图谱**：岗位关系可视化

## 部署建议

### 开发环境
- 使用 `npm run dev` 启动开发服务器
- 确保后端服务运行在 `http://localhost:8000`

### 生产环境
- 使用 `npm run build` 构建生产版本
- 部署到静态文件服务器（如Nginx、Apache等）
- 配置反向代理指向后端API

## 常见问题排查

### 1. 依赖安装失败

- 确保Node.js版本符合要求（^20.19.0 || >=22.12.0）
- 尝试清除npm缓存：`npm cache clean --force`
- 重新安装依赖：`npm install`

### 2. 后端连接失败

- 确保后端服务已启动
- 检查API请求地址配置
- 验证网络连接和CORS设置

### 3. 构建失败

- 运行类型检查：`npm run type-check`
- 检查TypeScript错误
- 确保所有依赖已正确安装

### 4. 页面空白或报错

- 检查浏览器控制台错误
- 验证路由配置
- 确保后端API响应正确

## 维护注意事项
