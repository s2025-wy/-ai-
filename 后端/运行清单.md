# 后端运行清单

## 项目概述

本项目是基于AI的大学生职业规划智能体的后端服务，使用FastAPI框架构建，提供用户认证、学生档案管理、岗位搜索、职业规划、智能问答等功能。

## 技术栈

- Python 3.10+
- FastAPI (Web框架)
- SQLite (数据库)
- Neo4j (图数据库)
- LangChain (LLM集成)
- DashScope API (大模型)
- Pydantic (数据验证)
- JWT (身份认证)

## 环境配置

### 1. 安装Python

确保安装Python 3.10或更高版本。

### 2. 安装依赖

```bash
# 进入后端目录
cd 后端

# 安装依赖
pip install -r requirements.txt
```

### 3. 配置环境变量

创建 `.env` 文件（如果不存在），添加以下配置：

```env
# JWT密钥
SECRET_KEY=your_secret_key_here

# Neo4j配置
NEO4J_URI=bolt://localhost:7687
NEO4J_USERNAME=neo4j
NEO4J_PASSWORD=your_neo4j_password

# DashScope API密钥
DASHSCOPE_API_KEY=your_dashscope_api_key

# 数据库配置
DATABASE_URL=sqlite:///./database.db
```

### 4. 启动Neo4j数据库

确保Neo4j数据库已启动并运行在默认端口7687。

## 运行步骤

### 方法一：使用start.bat脚本（Windows）

1. 双击 `start.bat` 文件
2. 服务将自动启动在 `http://localhost:8000`

### 方法二：手动运行

1. 进入后端目录
2. 执行命令：
   ```bash
   python main.py
   ```
3. 服务将启动在 `http://localhost:8000`

## 常见问题排查

### 1. 端口占用

如果8000端口被占用，可以修改 `main.py` 文件中的端口配置：

```python
if __name__ == "__main__":
    uvicorn.run(app, host="0.0.0.0", port=8001)  # 修改端口为8001
```

### 2. Neo4j连接失败

- 确保Neo4j服务已启动
- 检查 `.env` 文件中的Neo4j配置
- 验证Neo4j用户名和密码是否正确

### 3. DashScope API错误

- 确保 `DASHSCOPE_API_KEY` 配置正确
- 检查API密钥是否有效
- 验证网络连接是否正常

### 4. 数据库错误

- 确保SQLite数据库文件有读写权限
- 检查 `database.py` 中的数据库配置
